Initial public release of Pocket - a local-first database for web applications.

Features:
- Core database engine with collections, documents, and reactive queries
- React hooks for seamless integration (useLiveQuery, useDocument, useMutation)
- Sync engine for client-server synchronization
- Multiple storage adapters: IndexedDB, OPFS, and in-memory
- Schema validation with runtime type checking
- TypeScript-first with full type safety

## [0.1.0] - 2026-01-18

### Added

- Initial release
- OpenAPI 3.0/3.1 parser with full $ref resolution
- GraphQL schema parser
- TypeScript types generator
- Zod schemas generator with runtime validation
- HTTP client generator with fetch API
- React Query hooks generator (v5)
- SWR hooks generator
- CLI with `generate` and `init` commands
- Watch mode for development
- Configuration file support (`bridge.config.ts`)
- Tree-shaking friendly ESM output
